在王面前见证 Witness Before Kings

发表于 2015-05-31 | 分类于 使徒行传

徒 Act 25章

基督教的核心信息是永不改变的，在不同的时间和空间都是一致的！无论是在保罗的第一次，第二次，第三次的宣教旅程中，还是在去罗马的上诉旅程中，这个信息都是一致的，就是耶稣的死而复活。我们也看到，无论保罗在雅典的哲学家面前，还是在以弗所的迷信拜偶像的民众面前，或者在公会的宗教领袖面前，又或者在巡抚腓力斯、非斯都和亚基帕王面前，保罗所传讲信息的核心都没有改变，是一致的，就是耶稣的死而复活。

而保罗如此不厌其烦的在巡抚腓力斯、非斯都和亚基帕王面前传讲，是因为他确信，他们都需要耶稣基督的救恩。虽然在人看来他们位高权重，生活无虞，什么都不缺乏，但是他们若没有耶稣，他们就是失丧沉沦的！

主祷文——免我们的债，如同我们免了人的债

发表于 2015-05-24 | 分类于 马太福音

太 Mat 6:12 免我们的债，如同我们免了人的债

马上来到主祷文系列的第8讲，三求中的第二求，关于神是赦免者。

这节经文同样在路加福音11:4节中出现，那里说：赦免我们的罪，因为我们也赦免凡亏欠我们的人。这节经文容易给人一种误解，好像是是在告诉我们需要先赦免人的罪，然后才能得神的赦免，真是这样的吗？如果是这样，就是在说神赦罪的恩典不是无条件的，但圣经明明告诉我们得救是本乎恩也因着信，那我们要怎样理解这段经文呢？是作者笔误吗？是耶稣的教导不够准确吗？这次，我们将一起从圣经的文法、上下文以及上帝的饶恕与人的饶恕之间的关系来讲解这句祷文，盼望神祝福他的话语，愿神得荣耀。

神国与世界无法兼容（下）

发表于 2015-05-17 | 分类于 约翰一书

约一 1Jn 2：15-17

什么是“世界”？就是在神的国以外，不顺服神、抵挡神的由撒旦所统治的地方。

使徒彼得说你们“务要谨守、警醒，因为你们的仇敌魔鬼，如同吼叫的狮子，遍地游行，寻找可吞吃的人。”世界对付基督徒是不客气的，它会用各种手段来俘虏、吞吃那些软弱的基督徒。那么世界是用什么方式来吞吃基督徒的呢？人从最基本的需求开始犯罪，但他不会只停留在吃吃喝喝的层面，人的罪性会膨胀，让他不断地确信自己的能力，像吸毒一样会上瘾，他会沿着这个方向一步步走向骄傲，直到以为自己象神一样有智慧，并成为自己的主宰。

那么我们怎样才能对付魔鬼的试探呢？主耶稣为我们做了最好的榜样。只要我们住在这地上的帐棚，撒旦就会不断攻击我们，我们没有别的出路，我们只能向神谦卑地承认我们是软弱的，我们始终只能仰望那创始成终的耶稣基督。圣灵告诉我们“这世界和其上的情欲都要过去，惟独遵行神旨意的，是永远长存。”

抓住救恩 Salvation is Now

发表于 2015-05-10 | 分类于 使徒行传

徒 Act 24:1-27

圣经告诉我们巡抚腓力斯是个知道真道的人，而且是很详细清楚的知道，不是道听途说。

腓力斯为什么一拖再拖，在圣经中我们至少看到两个原因：

首先因为他想从保罗受贿（v26）

其次因为他想讨犹太人的喜欢（v27）

腓力斯和我们每一位都必须要面对将来的审判，耶稣基督不仅是人类的救主，也是人类的审判者，这是他第一次和第二次再来的时候不同的角色。腓力斯不单是理智上详细知道这道，而且情感上也有反应，他的内心被恐惧所触动，然而他仍然不肯作出一个意志上的决定，不肯遵行真理。腓力斯的选择是什么呢？仍然是拖延，在这章经文中重复提到他的拖延。22节说”支吾”他们，新译本译作故意拖延他们，25节再次的拖延做决定。

相信耶稣基督的最佳时刻就是现在！如果你还没有接受耶稣，我邀请你认真看待今天这个机会！
