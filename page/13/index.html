虚假的平安 False Peace

发表于 2015-08-30

结 Eze 11:1-25

作为一枚吃货，我想很多人都喜欢吃火锅，尤其是四川、重庆来的弟兄姐妹。在今天我们要看的经文中，有些人（圣城耶路撒冷的领袖）把自己比作锅里的肉，把耶路撒冷城比作锅，这是什么意思呢？

这个比喻可能是最让我们中文读者费解的了，这锅和肉很容易让我们想到”吃“。作为一个吃货的民族，我们关心的是这锅里的肉到底要怎么做？是做红烧肉、腊肉，还是做肉汤？但不管怎么样，最后的结局都是被吃掉。中国人有句成语叫做“人为刀俎，我为鱼肉”，比喻生杀大权掌握在别人手里，自己处在一个任人宰割的地位。那这也是古代犹太人的所要表达的意思吗？如果我们如此理解的话，意思就变成了这些领袖的危机感很强，觉得他们就要被巴比伦宰割了，那他们现在开的秘密会议就会类似电影《十月围城》里面的情景，讨论的是应该如何救亡图存的事了！

然而，这并不是今天我们要看的经文的原意，经文的原意可能要颠覆/逆转你对这经文的认识，也即是说，经文要表达的意思可能与你所想的恰恰相反。所以，有时候如果我们以我们固有的思维模式来读圣经真的会谬以千里，甚至闹出笑话。那么，领袖们说的这句话究竟是什么意思呢？意思乃是耶路撒冷城已经造假好了，已经固若金汤了，他们非常的骄傲的说出第三节的话，耶路撒冷城坚固得就像一个钢精锅，而他们这些领袖就像是锅里的肉那样安全，被锅保护着，不会被天空的飞鸟叼走。在一些比较新近的中文译本中，为了避免中文读者的误解，把这种解释也放进了翻译里面。比如我手头上两个比较现代的中文译本是这样翻译的：CNLT（新普及译本）译作”这城就像一个铁锅，我们在当中就像肉在锅里一样安全“；TCV（现代中文译本）译作”这座城像锅，我们像里面的肉，得到锅的保护“。

在今天的经文中神要摧毁耶路撒冷领袖们这种虚假的安全感……

没有最恶，只有更恶

发表于 2015-08-23

结 Eze 8:1-18

常常我们在网络上看到各种媒体事件不断的刷新人们对道德观，对人性的底线的认识！各种无耻的秀下限，智商的秀下限！我们今天以为这件事情很离谱，明天你会发现还有更离谱的事情。

实际上在整个人类历史当中，人是都是这样。没有一个最恶的时代，也没有哪一个民族是最恶的民族，整个人类历史就是一个攀比谁更邪恶的比赛！

在以西结书8章当中，神透过一个异象也让先知以西结更清楚的看见他的百姓以色列亦是如此！

这个异象发生的时间，是在约雅斤王被掳第六年的六月初五日（公元前592年9月17日），也就是在先知以西结刚刚完成他的430天躺卧大型行为艺术后不久。

这个异象当中，当先知以西结被树立在圣殿门口的偶像震惊的时候，神说，如果你以为这就是最可憎的事，你就太幼稚了，我再带你进里面去看看更刺激的！在接下来的参观圣殿异象之旅中，神一次又一次地刷新了以西结对百姓罪恶认识的下限！神对先知所说的话当中，最频繁出现的就是两句话：人子啊，你看见吗？（4次，6，12，15，17节）；然后就是“你还要看见更大可憎之事”（3次，6，13，15节），这也是我们今天要分享的核心内容！

不叫我们遇见试探

发表于 2015-08-16

太 Mat 6:13

不知道你祷告的时候是否有一种体验，当你经常为同一件事情祷告的时候，你发现祷告开始乏味，迫切的心也开始乏力。那么，‘不叫我们遇见试探，救我们脱离凶恶’就是这样一个我们经常做的祷告，在今天和下一次的讲道里，我会仔细和大家一起来分享关于这节经文的祷告。

这次，我们主要分享前半句‘不叫我们遇见试探’。当你看到这个祷告的时候，你会不会反问主耶稣：我们在这个世代，试探太多了，怎么才能让我们不遇见试探呢？ 可能唯一不遇见试探的方法就是被主接走了，那么主耶稣在这里是不是教导我们要为自己祷告要早点上天堂呢？这里主耶稣显然不是这个意思，生有时、死有时，凡事都有定时，都在神的时间。那怎样才能不叫我们遇见试探呢？是不是让我们远离红尘成为沙漠教父呢？当你认真阅读整个新约，你就会发现这也不是主耶稣的意思，那么为什么主耶稣要教导我们祷告“不叫我们遇见试探”？我们人人都知道，试探无处不在，尤其是在人群拥挤的城市，上班有上班时的试探，有时候上班本身就是一种试探，休息有休息时的试探，群居的时候有试探，独处也逃不出试探的降临，既然试探无处不在，耶稣的这句祷文是不是就白费口舌，是无法实现的梦想呢？

这次我们就一起来看看什么是试探，它和试炼有什么区别，试探来自哪里？我们如何面对和胜过试探。

以西结蒙召 Ezekiel’s Calling

发表于 2015-08-02

结 Eze 1-3章

最近一个多月，当临近使徒行传系列讲道完结的时候，我的心一直都在纠结，接下来要讲哪一卷书呢？

在以前鲜有用以西结书来讲道，这也说明这是一卷不容易传讲的书，而这种困难很多时候是来自心理作用：牧者怕会众不够成熟，不能深刻明白以西结书所传讲的信息，或者传讲起来会太枯燥——这是许多旧约先知书给人的印象；而大多数会众对旧约先知书有一种天然的望而生畏，不是觉得太晦涩了，就是觉得读起来太枯燥，一下子就跳过了，或者觉得那个信息充满了审判，没有我们想要听的慈爱和盼望的信息，我们自然人肉体里面的一些东西让我们无理由的抵抗先知书。

但是，我相信神的话语是完备的，没有一卷书是多余的，整本圣经（包括以西结书）都是神所默示的，都应当被传讲！

所以，今天我正式邀请大家和我一起开始这场极为特别的冒险之旅。不要怕，只要信，相信圣经都是神所默示的，于教训、督责、使人归正、教导人学义都是有益的，叫属神的人得以完全，预备行各样的善事。
