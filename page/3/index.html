求你以怜悯为怀

发表于 2016-11-27 | 分类于 哈巴谷书

哈巴谷书 Hab 3:1-16

哈巴谷书以一篇祷告的诗歌作为结束，从这首诗歌内容来看，较为圆满的解决了先知的经验与神学两方面平衡的问题，先知从最初对上帝的启示（迦勒底人灭犹大）是质疑的，或者说不顺服的，哈巴谷先知无论他多么不愿意相信，无论他怎么和神辩论，但是到了第3章他最终接受神的安排了并且学会了静默等候神。当我们读这首诗歌的时候，我推荐要从这三方面入手，第一，看它优美的文字，第二，看它要向我们传达关于上帝什么样的信息，第三，要看它具有什么样的感情色彩。

那么上次分享的1~16节这段祷文传达出怎样的信息呢？第一，表达出哈巴谷对神启示内容的完全顺服，并且他希望这天早点来到，他说：‘求神这些年间显明出来，复兴你的作为，什么作为呢？后面说：我只可安静等候灾难之日临到、犯境之民上来’；第二，神是有威严和权能的神，先知哈巴谷对神非常的敬畏，因为他认识神极其威严和有能力，先知用了大部分篇幅来赞美神的威严权能；第三，他是爱以色列民的，因为他祈求说，求神在发怒的时候以怜悯为念。

按先知曾经的信仰经验，上帝不应该借着一个更加邪恶的民族来击打他的选民，但是经过向神的哭诉、质疑、祷告之后先知转变了，你认为是什么促成了先知的转变呢，是什么让先知选择了顺服神的启示呢？在这个祷告诗篇中可以寻到一些答案，那就是在先知眼中，神是非常真实的。先知说：‘我听见你的声音，惧怕你的作为’，他听到了神的启示，而且先知见过神的作为，仅仅这样就说明了神的真实吗？不一定，对于今天的我们，很多信徒的问题是神只是停留在大脑（理性）里，没有进到心里，人性的问题是人性本身，人性的答案却在人性之外，救恩不是从人类中间产生的，只有超越的神来到你的里面，你才会被改变，所以，当法利赛人问耶稣：‘神的国几时来到呢？’，耶稣怎么回答？耶稣回答说：‘神的国来到、不是眼所能见的．人也不得说、看哪、在这里．看哪、在那里．因为　神的国就在你们心里’。只有那位在你头脑（理性）里的真神，进到你的心里，这位神对你来说才算是真实的，所以，先知哈巴谷才会得出约伯一样的结论：‘我从前风闻有你，现在亲眼见到你’。先知不但认为神是真实的，他更对神的伟大、威严和权能也大发赞美，这是他对神属性认识的告白，我们要向先知一样既要让神进到我们心里，同时也需要多多认识神的属性，神进入心里，可以让我们顺服，正确的认识神可以让我们产生正确的顺服。

此外，从这段诗文我们还可以看出哈巴谷先知对以色列有着强烈的热爱和怜悯，因为他祷告说：‘耶和华阿、求你在这些年间复兴你的作为、在这些年间显明出来、在发怒的时候、以怜悯为念。’上帝拣选以色列人，要在这个世界做他的代表，彰显他的作为，但是以色列人失败了，于是上帝亲自差派他独生爱子来到世界，借着使徒们开始建立教会，这次上帝拣选了教会，以色列国预表着今天的教会，那么谁又是这个时代的哈巴谷呢？是周牧吗？是师母吗？还是教会同工？我们每一个人都是，我们现在和旧约时代的以色列人有所不同，我们是已经看见或听见了基督的救恩，他们是远远盼望着这个救恩，旧约里只有很少一部分在不多的时段接受圣灵的启示，但是如今的我们有圣灵时刻的同在，我们每个人都可以透过耶稣基督和神建立了直接的关系，我们每个人都承担着向先知哈巴谷一样的爱着教会的使命，当教会陷入争端的时候，我们要爱教会；当你发现教会有很多罪恶的时候，我们要爱教会；当我们和牧者意见不同时，我们要爱教会；就算教会的领袖跌倒了，我们仍然要尊重牧者，为着神的缘故去爱领袖、爱教会。只要你是教会的一员，那么你就是基督身体的一个肢体，就应该像哈巴谷先知一样，在教会犯罪的时候，为罪痛悔，求主复兴，在教会受到外部冲击的时候，求主以怜悯为怀，用爱心彼此联络，建立基督的身体。

所以，今天的我们再次打开哈巴谷这个诗篇的时候，求圣灵再次打开我们的心，在面对上帝的启示和命令的时候，我们向先知哈巴谷一样，因着神是真实的，并且我们看到上帝的伟大而顺服他，同时，我们也有先知一样，对教会爱的胸怀，因为我们是被拣选的族类、是有君尊的祭司、是圣洁的国度、是属神的子民、要叫我们宣扬那召我们出黑暗入奇妙光明者的美德。

十角兽 A Ten-horn Beast

发表于 2016-11-20 | 分类于 但以理书

但以理书 Dan 7:15-28

我们简单回顾一下，上次的讲道我们看到了这个异象的2个场景或舞台，一个是来自恐怖混乱的海，一个是来自荣耀和谐的天上，海上上来了四个骇人的兽，天上有一个宝座，坐着亘古常在者，就是父神的位格，还有千千万万的天使，然后有一位像人子的被带到亘古常在者面前。我们说他就是圣子的位格。我们看到在古代近东的世界观中，海是混乱无序，具有破坏性力量的，是与神及其受造物为敌的。

但以理见到在这个异象，心里非常愁烦，甚至惊恐万分，绝对是个噩梦，就在这个梦中，但以理问一位神的侍者，求他解释这个异象，这些兽代表什么，大部分我们在上周的讲道中我们已经解释过了。 第四兽是谁？十角是谁？那个最后的小角是什么？一载，两载，半载又是多久？这些都是千百年来困扰人们的难题，我也不要给大家一个明确的答案，因为这个不是核心的信息。这里我想有必要说明一点的是，我们必须承认我们的有限以及神旨意的隐秘性。不幸的是，许多注释书，尤其是时代论主义立场的注释书常常把我们带往错误的方向，要么太关注细节，从而导致臆测，要么从错误的起点出发，从而扭曲了诠释结果。一位有见地的神学家警戒我们，不要试图“驯服深奥的启示文学”。

纵观第七章，要给我们的核心信息乃是，在最后的审判中，神会毁灭最后一位敌对神的统治者，然后将永远的国交给神圣的人子和神的圣民。作者这一信息的目标，不是为了满足21世纪人们对于末后日子的好奇心，神的子民正远离应许之地，在被掳中遭受折磨，他们感到被神抛弃，并逐渐完全丧失希望，但以理在这一信息中的目标，是为了向神在被掳中的子民保证，我们的至高神掌管这些恶的人间国度，最后并要借着一位神圣的人子，并他的国度交给他的子民。

联想到今天的美国或者中国，有些基督徒老是有一种政治幻想，要借着人间的君王来实现圣经的价值观。中国的基督徒喜欢穿一些什么中南海也有查经班，温家宝，杨利伟是信主的，彭丽媛唱哈利路亚是颂赞神之类的谣言和自我陶醉的幻想，而美国基督徒则将它们的梦想托付在共和党手中，好像川普上台美国就变成我主和主基督的国了，而希拉里上台则是启示录中的大淫妇，是耶洗别的灵。当然作为一个地上国度的居民，在神的普遍恩典之下，我们享有这样一个公民参与政治的民事权利，这是无可厚非的，基督徒不是消极避世的人。但是希望透过但以理书的信息让我们看见这神的国，我们的国不属于这世界，无论是民主国家还是独裁国家，无论是公义的国家还是邪恶的国家都是一只兽，不是人子，无论是希特勒，斯大林，金三胖还是丘吉尔，朴槿惠，奥巴马都在神的掌管之下，最终都要向神交账，因为天上的案卷已经展开。

启示录最后告诉我们，那一天，兽也不会有了，海也不再有了，有大声音从神的宝座出来说：看啊，神的帐幕在人间。那一天，真正属于我们的国将会来到，不是我们选票投出了的，而是从天上降下的。这个世界的国在行使权力时所导致的混乱，不是最后的结局，天上的审判者要坐着审判历史中的兽们，他们要被毁灭，而忠信的圣徒会得到治理权和统治权。对被掳中的民族，对今天忍受痛苦逼迫的人，或是对今天生活在暴力，流血和腐败中，属神的忠信百姓来说，这是最能给人盼望的信息。

魔兽世界 A Beasts Vision

发表于 2016-11-13 | 分类于 但以理书

但以理书 Dan 7:1-14

本周最引人关注的事件莫过于美国大选了，出乎所有人的意外，川普赢得了选举。而从这半年多来，各大主流媒体都是不看好川普的，几乎都是一边倒的支持希拉里。让我想到最近我们一直分享的但以理书都是关于异象异梦，还有王朝的更替。但是但以理书同样也提醒我一件事，那就是神才是终极的掌权者，至高者在人的国中掌权，他要将国赐给谁就将国赐给谁。请不要误会，我绝对不要要引用圣经为川普背书，支持他的意思。你要记住但以理所记载的王中，可没有一个是好人，只有非常恶和一般恶。C.S. Lewis也提醒我一件事，对于这两个候选人，一个是疯子，一个是骗子，但是感谢主，神赐给我们的王，耶稣他既不是疯子也不是骗子，而是我们的主和救主！

如果我们说但以理书前六章是发生在物理世界的事，那么但以理书后六章就是发生在灵界的事了。今天这一章是灵界启示的开始。

这一章基本可以分为3个大块，第一部分（1-8）的舞台是海，从海中上来四个兽，第二部分（9-14）的舞台是天上，那里有神的宝座，和神的2个位格，第三部分（15-28）神和但以理的对话，解明这一切的事。前两个部分是同一个异象的不同场景，第三部分是神将异象的解释启示出来。

我们看到这四个兽的异象中，四个兽各不相同，代表不同的国。即使在今天有些国家也用动物来代表，比如美国是鹰，英国是狮子，俄罗斯是熊等等。但是但以理看到的绝非这些正常的动物，而是一种混杂性的动物，是异形动物。这些异形的动物是秩序的破坏和写照，它代表了邪恶的，充满恶意的人类之国。

圣经让我们看到人间的权柄最终都要被收回，交给那位像人子的，使各方、各国、各族的人都事奉他。他的权柄是永远的，不能废去；他的国必不败坏。

人间的权柄最终都会被收回，无论是尼布甲尼撒，伯沙撒，大利乌还是亚历山大，无论是毛泽东或习近平，今天一半的美国人开心，另一半的美国人伤心欲绝，这个国家被撕裂，他们集会示威，打出横幅“川普不是我们的总统”，他们不承认这个权柄。

默想基督

五百年后，耶稣基督在这地上行走的时候，谦卑的自称为人子，当大祭司起来问耶稣，你是那当称颂者的儿子基督不是？耶稣回答说，我是，你们会看见人子坐在那全能者的右边，驾着天上的云而降临。他明明白白引用大祭司熟悉的但以理书来宣告自己的神性！

耶稣也将他的再临与末日的灾难联系起来，耶稣说，在那些日子，那灾难后，那时，他们要看见人子有大能力，大荣耀，架云降临，他要差遣天使，把他的选民从四方，从地极直到天边，都招聚了来。

但以理书说：这位像人子的得了权柄、荣耀、国度，使各方、各国、各族的人都事奉他。而耶稣复活升天之后，他说天上地下所有的权柄都赐给我们了，所以你们要去使万民做我的门徒！

封闭的狮子坑 A Closed Pit

发表于 2016-11-06 | 分类于 但以理书

但以理书 Dan 6:13-28

上周我们分享到一个敞开的窗户，今天我们要接着分享一个封闭的狮子坑。

对于但以理时代的读者而言，被掳中的以色列百姓会认同但以理，因为他们也生活中外邦的律法之下，但以理被扔进狮子坑，他们也正处于被掳的“坑”中，因此，但以理的故事鼓励他们持守对神的信心，相信神能拯救他们。今天对于被掳于世界这个“大狮子坑”中的我们，撒旦就是那遍地游行的狮子，要吞噬一切世上的灵魂，他每天对我们张牙舞爪，要把我们撕碎，我们怎样才能幸免于难呢？怎样才能在仇敌的权势下存活呢？只有那些投靠在权柄大过仇敌撒旦的，持守对永生神的信心。相信他必保守我们脱离狮子的口，神绝对有能力保守我们在世界这个大坑中安然无恙。

从终极的意义来看，死亡就是那个最终的大坑，我们都将进入的一个大坑，但是这坑不是结束，这坑是一个中转站，不是终点站，这坑是见证神大能拯救的地方，因为我们相信，有一天，神会将我们从这坑中提起，提到他的宝座前，神将成为我们的审判者，就是“但以理”这名字所启示的，我们要进入他无比的荣耀中，我们在坑中不过是一宿，但在荣耀中是无限的永恒！

我们之所以可以如此，是因为有一位本来不属于这坑的人，他下到这坑当中，他不仅下到这世界的大坑中，他也进入到死亡的大坑中，一位天上的神来到了地上，一位不会死的神经历了真实的死亡，目的是为了要拯救我们这些本该死的人，使我们这些本该死的人可以脱离死亡的坑！

但以理明显地预示了基督，但耶稣的伟大远非但以理可比，但以理面临死亡的威胁，而耶稣却是经历死亡；但以理脱离了他的狮子坑，最终仍会死；耶稣从他的坟墓出来，却永远活着。从狮子坑中被救出的但以理在大利乌在位期间，大享亨通，但从坟墓出来的耶稣，神将他升为至高，又赐给他那超乎万名之上的名，叫一切在天上的、地上的，和地底下的，因耶稣的名无不屈膝，无不口称耶稣基督为主，使荣耀归与父神。

最后，神将但以理从狮子坑中救出，从而给被掳的以色列百姓希望，神会从被掳的“坑”中拯救他们，但神使他的儿子耶稣基督从死里复活，这给了我们更大的盼望，神也会就我们的身体和灵魂脱离死亡，从死里复活，正如保罗所说，基督已经从死里复活，成了睡了之人初熟的果子！
