小组建造 Cell Group Building

发表于 2014-01-05 | 分类于 以弗所书

证道/周牧

弗Eph 4:12-13

旧约中的小组

  • 最初的小组 创一：26

  • 出埃及时代的小组 出十八：24-26
    新约中的小组

  • 十二门徒的小组 可三：13-15

  • 初代教会的小组 徒二：46
    小组的益处
  1. 彼此鼓励并分享基督的爱（加拉太书五：13；以弗所书五：2）。
  2. 强调合一（使徒行传四：32；罗马书十二：5）。
  3. 建立基督的身体（歌罗西书二：6-7）。
  4. 开发属灵的恩赐（罗马书十二：6-8；哥林多前书十二：4-11）。
  5. 教导真理并培养门徒（提摩太后书二：2）。
    小组建造
  • 全教会信徒都需要过小组团契的生活
  • 小组是教会团契生活最有效的结构
  • 建造小组团契的教会所运用之原则
  • 经常性的聚集徒2：44，46
  • 一心一意徒4：32
  • 分享精神 徒2：44-45
  • 需要得到满足 徒4：33-35
