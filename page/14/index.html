审判列国的神 God Who Judge Nations

发表于 2015-09-20 | 分类于 以西结书

以西结书Eze 25-32

今天是一个高度全球化的时代，没有一个国家是一座孤岛，可以独立存在，不仅是今天，一直以来以色列与环绕他四围的国家的爱恨情仇都是这个世界重要的主题，更加是圣经的主题！今天我们要来看看神审判以色列四围列国的信息!

感谢主，今天我们终于开始了以西结书的后半部分，如果说前24章是对以色列的审判，那么后24章就是对以色列的的应许和盼望了。其实如果我们留意到的话，其实前面24章每个审判当中都带着盼望。而这后24章又是以8章的审判神谕开始的，但是这个审判，却是对以色列的盼望，因为是审判列国，就是那些以色列的仇敌。而这8章的审判，又可以分为2部分，前4章是对六国的审判神谕，后4章是对埃及的审判神谕。而夹在这2个部分之间的短短3节经文却是赐给以色列盼望的话语，无论是从文本位置上，还是内容上都可以称得上是这8章经文的核心。

神审判列国的目的，就是要拯救他的百姓以色列！

父债子偿？Father's Fault？

发表于 2015-09-13 | 分类于 以西结书

以西结书Eze 18:1-32

小时候读武侠小说，或者看武侠连续剧的时候，常常出现的口头禅就是“父债子偿，血债血偿，冤冤相报何时了，君子报仇十年不晚”之类的俗语，表达了一些类似的意思，就是一个人犯的错，却由他的后代来偿还。

其实类似的俗语在古代以色列百姓当中也存在：父亲吃了酸葡萄，儿子的牙酸倒了（结18：2）。这样的俗语给了以色列百姓一些虚假的安慰，神正是要透过先知以西结摧毁这种虚假的安慰，让他们直面自己的光景！

其实我们也是一样，当人们处在一个困境当中的时候，常常会想到去抱怨各种原因，父母的原因，环境的原因，别人的原因，老板的原因，却常常不去反省自己有什么问题……

无可替代 No Substitution

发表于 2015-09-06 | 分类于 以西结书

以西结书 Eze 14:12-23

“虽有这三人在其中，主耶和华说：我指着我的永生起誓，他们连儿带女都不能救，只能因他们的义救自己的性命”

最近流行一句话，叫作“重要的话说三遍”，但是今天的经文中我们看到有一句话神说了四遍，足见这是多么重要的一句话或者一个真理。要认识这个真理，我们就需要知道这件事情的背景！

在以西结书前面13章当中，神让先知发预言责备当时那些以色列中的假先知，而14章的前半章当中提到那些假先知被迷惑说一些虚假的预言。百姓也听从了这些虚假的预言而产生虚假的盼望。14章一开始有一些长老来求问先知以西结，于是以西结就向这些长老们发了预言，这也是今天我们要分享的内容。在上一篇讲道中，我们看见神借着先知以西结摧毁耶路撒冷领袖虚假的平安，在今天的经文中神借着先知要摧毁被掳的百姓对耶路撒冷虚假的盼望，神说耶路撒冷绝对不能躲过他的审判！

虚假的平安 False Peace

发表于 2015-08-30 | 分类于 以西结书

结 Eze 11:1-25

作为一枚吃货，我想很多人都喜欢吃火锅，尤其是四川、重庆来的弟兄姐妹。在今天我们要看的经文中，有些人（圣城耶路撒冷的领袖）把自己比作锅里的肉，把耶路撒冷城比作锅，这是什么意思呢？

这个比喻可能是最让我们中文读者费解的了，这锅和肉很容易让我们想到”吃“。作为一个吃货的民族，我们关心的是这锅里的肉到底要怎么做？是做红烧肉、腊肉，还是做肉汤？但不管怎么样，最后的结局都是被吃掉。中国人有句成语叫做“人为刀俎，我为鱼肉”，比喻生杀大权掌握在别人手里，自己处在一个任人宰割的地位。那这也是古代犹太人的所要表达的意思吗？如果我们如此理解的话，意思就变成了这些领袖的危机感很强，觉得他们就要被巴比伦宰割了，那他们现在开的秘密会议就会类似电影《十月围城》里面的情景，讨论的是应该如何救亡图存的事了！

然而，这并不是今天我们要看的经文的原意，经文的原意可能要颠覆/逆转你对这经文的认识，也即是说，经文要表达的意思可能与你所想的恰恰相反。所以，有时候如果我们以我们固有的思维模式来读圣经真的会谬以千里，甚至闹出笑话。那么，领袖们说的这句话究竟是什么意思呢？意思乃是耶路撒冷城已经造假好了，已经固若金汤了，他们非常的骄傲的说出第三节的话，耶路撒冷城坚固得就像一个钢精锅，而他们这些领袖就像是锅里的肉那样安全，被锅保护着，不会被天空的飞鸟叼走。在一些比较新近的中文译本中，为了避免中文读者的误解，把这种解释也放进了翻译里面。比如我手头上两个比较现代的中文译本是这样翻译的：CNLT（新普及译本）译作”这城就像一个铁锅，我们在当中就像肉在锅里一样安全“；TCV（现代中文译本）译作”这座城像锅，我们像里面的肉，得到锅的保护“。

在今天的经文中神要摧毁耶路撒冷领袖们这种虚假的安全感……

没有最恶，只有更恶

发表于 2015-08-23 | 分类于 以西结书

结 Eze 8:1-18

常常我们在网络上看到各种媒体事件不断的刷新人们对道德观，对人性的底线的认识！各种无耻的秀下限，智商的秀下限！我们今天以为这件事情很离谱，明天你会发现还有更离谱的事情。

实际上在整个人类历史当中，人是都是这样。没有一个最恶的时代，也没有哪一个民族是最恶的民族，整个人类历史就是一个攀比谁更邪恶的比赛！

在以西结书8章当中，神透过一个异象也让先知以西结更清楚的看见他的百姓以色列亦是如此！

这个异象发生的时间，是在约雅斤王被掳第六年的六月初五日（公元前592年9月17日），也就是在先知以西结刚刚完成他的430天躺卧大型行为艺术后不久。

这个异象当中，当先知以西结被树立在圣殿门口的偶像震惊的时候，神说，如果你以为这就是最可憎的事，你就太幼稚了，我再带你进里面去看看更刺激的！在接下来的参观圣殿异象之旅中，神一次又一次地刷新了以西结对百姓罪恶认识的下限！神对先知所说的话当中，最频繁出现的就是两句话：人子啊，你看见吗？（4次，6，12，15，17节）；然后就是“你还要看见更大可憎之事”（3次，6，13，15节），这也是我们今天要分享的核心内容！
